# FoodWasteR
Repositório dedicado ao projeto FoodWasteR, desenvolvido em R para análise e previsão de desperdício alimentar em restaurantes. Criado como ferramenta de aperfeiçoamento pessoal em Engenharia de Dados, mas com potencial uso profissional em gestão de dados, automação e inteligência preditiva aplicada ao setor alimentício.
